# WealthSutra-AI

WealthSutra-AI is an innovative project aimed at providing AI-driven solutions for wealth management and investment strategies.

## Features
- **Automated Portfolio Management**: Streamlines investment processes using automation.
- **Data-Driven Insights**: Provides users with actionable insights based on their investment data.
- **User-Friendly Interface**: Easy to navigate and use for both novices and experts in finance.

## Installation
To install WealthSutra-AI, clone the repository and install the dependencies:

```
git clone https://github.com/nidhi-builds/WealthSutra-AI.git
cd WealthSutra-AI
pip install -r requirements.txt
```

## Usage
Run the application using:

```
python app.py
```
